# Disease Prediction Using Machine Learning

## Overview
This project predicts diseases using a **machine learning model** trained on a dataset with **132 parameters** and **42 disease types**. The project is implemented in a **Jupyter Notebook** for end-to-end workflow, including **EDA, preprocessing, training, and evaluation**.

## Features
- **Data Exploration**: Exploratory Data Analysis (EDA) for training and testing datasets.
- **Data Preprocessing**: Handling missing values, encoding categorical variables, and normalizing features.
- **Model Training**: Training a **Decision Tree Classifier** (and other models) for disease prediction.
- **Model Evaluation**: Evaluating performance using accuracy, precision, recall, and F1-score.
- **Prediction**: Predicting diseases for new patient data using the trained model.
